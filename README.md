## 🌟 Overview

**Smart Bharat** is a **GenAI-powered civic platform** that revolutionizes how Indian citizens interact with government services. Instead of navigating complex portals and bureaucratic procedures, citizens simply *talk* to our AI companion — in their own language — and get instant, personalized guidance.

Built for the **PromptWars × Devengers Hackathon 2026**, this platform leverages **Generative AI** to:

- 🤖 Simplify complex government information into plain language  
- 📋 Answer citizen queries about 50+ government services  
- 🎯 Recommend eligible schemes & subsidies based on user profile  
- 📄 Generate accurate document checklists for applications  
- 📢 Track public complaints in real-time  
- 🌐 Support all **22 scheduled Indian languages**  

---

## ✨ Features

### 🤖 AI Companion (Bharat AI)
An intelligent chat assistant that understands natural language queries and provides step-by-step civic guidance. Powered by Generative AI with context-aware responses.

| Capability | Description |
|---|---|
| 🗣️ Natural Language | Ask in any Indian language, get answers in the same |
| 📋 Document Checklists | Exact documents required for any government application |
| 🎯 Scheme Recommendations | AI matches you to eligible government schemes |
| 📢 Complaint Tracking | File and track grievances through pgportal.gov.in |
| ⚡ Instant Responses | 24/7 availability with sub-second response times |

### 🏛️ Government Services Covered
- **Identity & Documents** — Aadhaar, PAN, Passport, Driving License, Voter ID
- **Health & Welfare** — Ayushman Bharat, PM Jan Arogya Yojana
- **Education** — NSP Scholarships, RTE Admissions, Student Services
- **Land & Property** — Land Records, Property Tax, Mutation
- **Tax & Finance** — ITR Filing, GST, Income Certificates, Subsidies
- **Business** — Udyam Registration, MSME Schemes, Startup India
- **Grievances** — RTI Requests, Public Issue Reporting
- **Agriculture** — PM Kisan, Crop Insurance, Kisan Credit Card

### 🎨 Design Highlights
- **Dark Premium UI** with glassmorphism and animated particle background
- **Interactive Chat UI** with keyword-aware AI responses
- **Animated Canvas** background with floating gradient orbs
- **Mouse Parallax** floating cards on the hero section
- **Smooth Scroll Animations** triggered on intersection
- **Fully Responsive** — mobile, tablet, and desktop
- **Marquee Service Strip** for visual engagement

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No build tools required — pure HTML/CSS/JS!

### Run Locally

```bash
# Clone the repository
git clone https://github.com/RajSingh2006-git/GenAI-powered-web-platform.git

# Navigate to the project
cd GenAI-powered-web-platform

# Open in browser (macOS)
open index.html

# Or on Linux
xdg-open index.html

# Or on Windows
start index.html
```

That's it! No npm install, no build steps, no configuration needed. 🎉
